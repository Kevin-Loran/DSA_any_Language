# DSA Any Language

Repositório dedicado ao estudo de Estruturas de Dados e Algoritmos utilizando diferentes linguagens de programação.

## Objetivos

* Desenvolver raciocínio lógico e algorítmico.
* Aprender estruturas de dados fundamentais.
* Comparar implementações entre linguagens.
* Construir uma base sólida para entrevistas técnicas e desenvolvimento profissional.

## Linguagens

* C
* C++
* Java
* C#
* Python

## Estrutura

Cada pasta contém exercícios, implementações e anotações específicas da linguagem.
